# Departamentos

El Hotel Italia está organizado en los siguientes departamentos operativos, cada uno siguiendo el Marco Operativo MOHI:

- Recepción
- Housekeeping
- Restaurante
- Cocina
- Mantenimiento
- Administración
- Contabilidad
